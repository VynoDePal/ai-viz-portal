## Getting Started

### Sign Up

1. Visit the platform URL
2. Click "Sign Up" in the top right corner
3. Enter your email address and create a password
4. Verify your email address
5. Complete your profile setup

### Sign In

1. Click "Sign In" in the top right corner
2. Enter your email address and password
3. Click "Sign In" to access the platform

## Dashboard

The dashboard provides an overview of the latest AI model performance metrics.

### Key Features

- **Model Rankings**: Top performing models across benchmarks
- **Recent Benchmarks**: Latest benchmark results
- **Trends**: Performance trends over time
- **Quick Filters**: Access to common filters

### Navigation

Use the sidebar to navigate between different sections:
- Dashboard
- Rankings
- Trends
- Predictions
- Teams
- Settings

## Model Rankings

View and compare AI model performance across different benchmarks.

### Viewing Rankings

1. Navigate to the Rankings page
2. Select a benchmark from the dropdown
3. Models are ranked by their score on the selected benchmark

### Comparing Models

1. Select multiple models by clicking the checkbox next to each model
2. Click "Compare Selected"
3. View side-by-side comparison of model performance

### Filtering Rankings

Use the filter panel to:
- Filter by model name
- Filter by benchmark
- Filter by score range
- Sort by name, score, or date

## Trend Analysis

Track model performance over time with interactive trend charts.

### Viewing Trends

1. Navigate to the Trends page
2. Select a model and benchmark
3. The trend chart shows performance over time
4. Use the date range selector to adjust the time period

### Trend Features

- **Interactive Charts**: Zoom and pan to explore data
- **Annotations**: Mark important events and milestones
- **Comparison**: Compare multiple models on the same chart

## Predictions

View AI-powered predictions for model performance.

### Viewing Predictions

1. Navigate to the Predictions page
2. Select a model and prediction method
3. View predicted performance with confidence intervals

### Prediction Methods

- **Linear Trend**: Simple linear regression
- **Moving Average**: Smoothed trend analysis
- **Exponential Smoothing**: Weighted average with exponential decay

## Filters and Search

Use advanced filtering to find specific models and benchmarks.

### Search

Use the search bar to:
- Search by model name
- Search by benchmark name
- Search by keywords

### Filters

Apply filters to narrow down results:
- **Name Filter**: Filter by model name pattern
- **Benchmark Filter**: Filter by specific benchmark
- **Score Range**: Filter by minimum/maximum score
- **Date Range**: Filter by date range

### Saved Filters

Save frequently used filter combinations for quick access:
1. Apply desired filters
2. Click "Save Filter"
3. Enter a name for the filter
4. Access saved filters from the dropdown

## Team Collaboration

Work together with your team on shared workspaces.

### Creating a Team

1. Navigate to the Teams page
2. Click "Create Team"
3. Enter team name and description
4. Invite team members by email
5. Set member roles (Owner, Admin, Member, Viewer)

### Team Roles

- **Owner**: Full access to all team features
- **Admin**: Can manage members and settings
- **Member**: Can contribute to team projects
- **Viewer**: Read-only access

### Team Dashboard

View team activity and shared resources:
- Recent activity feed
- Team member list
- Shared filters and views
- Team statistics

## FAQ

### How do I reset my password?

1. Click "Sign In"
2. Click "Forgot Password"
3. Enter your email address
4. Follow the instructions in the email

### How do I change my email address?

1. Go to Settings
2. Click on "Account"
3. Update your email address
4. Verify the new email address

### How do I delete my account?

1. Go to Settings
2. Click on "Account"
3. Scroll to "Danger Zone"
4. Click "Delete Account"
5. Confirm deletion

### How do I contact support?

Email support at support@ai-viz-portal.com or use the contact form in the Settings page.

### How often is the data updated?

Data is updated in real-time using Supabase Realtime. You'll see changes as soon as they are made.

### Can I export data?

Yes, you can export data in various formats:
- CSV
- JSON
- Excel

Click the "Export" button on any page to download data.
